Infrastructure hook that drives the [HTML `popover` attribute](https://developer.mozilla.org/en-US/docs/Web/API/Popover_API) and [`<dialog>` element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dialog) from a reactive `open` signal. Used internally by Dialog, Popover, Menu, Select, Tooltip, etc. — these primitives render their content in the browser top layer, escaping parent overflow, transform, and stacking contexts.
